#FEITO PARA PROJETOS/ MINI PROJETOS
##PARA QUE USAR:
- Utilizado para fazer projetos utilizando todo o conhecimento aprendido durante a semana;
- Pode ser projetos grandes ou pequenos, mas eficientes para o aprendizado;
- Tambem pode ser projetos feitos anteriormente por mim ou outra pessoa;
- Utilizando para correção, atualização e/ou para otimização do projeto;

##REGRAS:
- Tem que usar arquitetura limpa;
- Revisar o código e tomar cuidado com a escrita;
- Código comentado e explicar o motivo de usar a função e separação de responsabilidade;
